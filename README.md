# Final Web Development Project

This repository contains a multi-page static website designed to satisfy the assignment requirements shown in the provided screenshot.

## Pages

- `index.html`: professional student career page with resume content, media, and a graph
- `scratch.html`: from-scratch interactive app built with custom HTML, CSS, and JavaScript
- `bootstrap.html`: separate Bootstrap-based page with additional custom styling

## Local use

Open `index.html` in a browser, or serve the folder with any static file server.

## Hosting

This project is ready to publish with any static hosting platform.
